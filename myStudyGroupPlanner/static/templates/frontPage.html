What is this Application?


It is a tool that allows you to form study groups with other students in your classes.


Features

* Join Multiple Groups

* Chat With Other Members of a Group

* Report Other Members in a Group


Coming Soon...

* Create a Profile Page

* Hire Tutors

* Select from Multiple Permission Levels

* Video Chat


This application is compatible with Google Chrome.
